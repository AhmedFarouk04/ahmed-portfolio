# Ahmed Farouk - Portfolio

A high-performance, fully responsive personal portfolio showcasing my experience as a Backend .NET Developer. Built with Vanilla JS, CSS3, HTML5, and bundled with Vite.

## 🚀 Features

- **Clean Architecture:** Modular structure dividing JS logic and CSS styles for optimal maintainability.
- **Custom Cursor & Smooth Scrolling:** Enhances UX with a tech-focused custom cursor and smooth reveal animations using IntersectionObserver.
- **Dynamic Project Filtering:** Instantly filter projects by technology (AI, .NET, Full Stack, Node.js).
- **Performance Optimized:** Bundled using Vite for lightning-fast HMR and minified production builds.
- **SEO Ready:** Pre-configured with Open Graph meta tags for optimal social sharing.

## 🛠️ Tech Stack

- HTML5
- CSS3 (CSS Variables, Flexbox, CSS Grid)
- Vanilla JavaScript (ES6 Modules)
- [Vite](https://vitejs.dev/) - Frontend Tooling
